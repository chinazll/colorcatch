# Color Catch - 糖果跳跳乐

## 概述
一款基于 HTML5 Canvas 的跳跃游戏，参考开心消消乐美术风格。玩家控制可爱的小糖豆在糖果色平台上跳跃，目标是跳上**颜色匹配**的平台。

## 核心玩法
- ← → 键控制左右移动（支持触摸拖动）
- 平台颜色 = 玩家颜色 → 匹配成功，得分 + 换色
- 平台颜色 ≠ 玩家颜色 → 游戏结束
- 橙色球 = Bonus，+50分

## 视觉特性
- **糖果色系**：红、黄、蓝、绿、紫、橙
- **圆润卡通角色**：大眼睛、高光、腮红、Q弹拉伸
- **粒子特效**：跳跃粒子、得分星星、消除爆炸
- **Q弹动画**：outBack/outElastic/outBounce 弹簧曲线
- **视差背景**：天空渐变 + 白云 + 山丘 + 草地
- **屏幕震动**：大Combo触发
- **崩溃平台**：踩上后0.6秒崩塌
- **移动平台**：左右自动移动
- **Bonus平台**：金色星星标记

## 技术栈
- 纯 HTML5 Canvas + Vanilla JS（单文件，无依赖）
- CSS3 动画（UI界面）
- Google Fonts: Nunito
